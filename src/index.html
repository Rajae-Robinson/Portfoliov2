ZenScholar

ZenScholar is a platform that supports university students struggling with depression, stress, and anxiety. This is a collaborative capstone project where I contributed to both research and implementation.

In leading the development effort, I chose to create a production-ready website utilizing React.js, Typescript, and TailwindCSS. The project includes Unit Testing, PWA support, and static analysis testing tools such as Husky, ESLint & Prettier in order to prevent regressions and enforce code quality during collaboration.

NASA Mission Control Demo Project

Create launches to visit one of the habitable exoplanets. Explore upcoming launches and delve into the history of past missions, including those by SpaceX.

This FullStack project is built using the MERN Stack (MongoDB, Express.js, React.js, Node), deployed on AWS, with CI/CD managed by GitHub Actions.

The robust backend employs REST API and MVC architecture, featuring token-based authentication (JWT), rate limiting, logging, error handling, and more.

When visiting the site, requests may be delayed due to cold starts if it hasn't been accessed in a while. Close the tab, reopen it, and try again.
